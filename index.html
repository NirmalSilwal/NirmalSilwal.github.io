    Some of my works

Generate Human Faces

Generate new (fake) images of human faces using Generative Adversarial Network (GANs) using CelebA Dataset of 200K celebrity faces. This project is Implemented in Pytorch with reference to Deep Convolutional Generative Adversarial Network (DCGAN) original paper.

Dog-Breed Classifier

Image classification project to identify the breed of Dog. If dog is detected it will predict breed of the dog. And if human is detected it will return resembling dog breed. It uses Convolutional Neural Network, both from scratch & using transfer learning (resnet50).

Deploying Sentiment Analysis Model

Fully operational AI project to do the sentiment analysis of the movie review. It uses IMDB dataset for training the model using Recurrent Neural Network (LSTM cell). It is developed on Pytorch & deployed on AWS Sagemaker.
